# URLs de Busca para Supermercados no Brasil

Para encontrar os melhores preços, utilize os seguintes padrões de busca no navegador:

## 1. iFood Mercado (Agregador)
- **URL**: `https://www.ifood.com.br/mercado/`
- **Procedimento**:
  1. Definir o endereço/cidade do usuário.
  2. Pesquisar o produto na barra de busca.
  3. Comparar os preços entre as diferentes lojas listadas.

## 2. Rappi (Agregador)
- **URL**: `https://www.rappi.com.br/`
- **Procedimento**:
  1. Definir o endereço.
  2. Ir para a seção "Supermercados".
  3. Pesquisar o produto.

## 3. Google Shopping (Local)
- **URL**: `https://www.google.com.br/shopping`
- **Procedimento**:
  1. Pesquisar por `[produto] [cidade] supermercado`.
  2. Filtrar por "Perto de você" se disponível.

## 4. Sites Oficiais de Nota Fiscal (Estaduais)
Estes sites mostram preços reais baseados em notas fiscais emitidas recentemente:
- **Paraná**: `https://menorpreco.notaparana.pr.gov.br/`
- **Rio Grande do Sul**: `https://menorprecobrasil.sefaz.rs.gov.br/`
- **Outros estados**: Pesquisar por "Menor Preço [Estado] Nota Fiscal".

## 5. Grandes Redes (Busca Direta)
- **Carrefour**: `https://mercado.carrefour.com.br/`
- **Pão de Açúcar**: `https://www.paodeacucar.com/`
- **Extra**: `https://www.clubeextra.com.br/`
- **Sonda**: `https://www.sondadelivery.com.br/`
- **Mambo**: `https://www.mambo.com.br/`
